# Security Policy

## Supported Versions

| Version| Supported          |
| -------| ------------------ |
| v0.1-dev|:x:|
| v0.2-dev-pre***x***|:white_check_mark:|

## Reporting a Vulnerability

1. Go to the issues tab of the project
2. Create a new issue
3. Write in the title "Security Vulnerbility"
4. Describe the security problem
5. Submit the issue
